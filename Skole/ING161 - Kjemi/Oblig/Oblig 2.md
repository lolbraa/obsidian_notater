$2 I _ { 2 } ( g ) + 2 H _ { 2 } S ( g ) \rightleftharpoons S _ { 2 } ( g ) + 4 H I ( g )$

Begrunn svaret.

```
Når K << 1 er det reaktantene som er favorisert ved likevekt.

```
